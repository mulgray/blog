Mul. Run as Mode IM With Syn RM

Aug 12th, 2005

今日も計画通りだ。

ただ気になるのは多少集中力が下がりかかっていることである。

いつかのリゲインの効果については、結論としては、何も無くても十分元気な時に補助として飲むと最大の効果を発揮しやすくなるといったところだろう。残念ながらBlackBlackほどには使い勝手は良くない。何故なら高いからだ。315円くらいで100mlである。

でも流石にサンクチュアリーの内側ではこれ以上集中力が下がることはない。上がることはあっても。自分自身にさえほとんど邪魔されることがない。

結局リゲインは無くても平気だった。当然と言えば当然だが、しかしスタートアップにはやはり貢献したと思う。

むしろリゲイン使用時の集中力が高すぎたとも言える。

今日でまた1タームを終えた講習は休み時間として受けていた。とはいっても別に気を抜いている訳ではない。自習より余程楽なだけだ。

最近ではこうして電車内で書き込むことがほとんどだが、ブログを始めた当初の目的の一つがそろそろ達成できてきた。タイピングの練習というものだ。今では人並みには打てると思う。大いに結構なことだ。

明日は兄の誕生日祝いをする。もう27歳になるのだ。

明日は私も休みを少しとるとしよう。

今日は何時にもなく文章をまとめられなかったが疲れているためだろう。この辺で今日は書き終わることにする。
